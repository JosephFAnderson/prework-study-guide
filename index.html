HTML

  • !DOCTYPE declartion tells the browser what type of document it should expect
  • The head element is a container for metadata
  • Meta elements contain information about the page that is used by the browser
  • The user does not see information in the head or meta elements
  • Types of information that can be stored in the meta tag
  • The body element represents the visible content shown to the user
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage
  • When possible, it is important to use semantic HTML elements to provide easier reading for other devolopers and the browser
  • Semantic HTML elements also make your webpage accesible to the widest variety of users, including those who use assistive technologies such as screen readers
  • Elements nested inside other elements are called child elements
